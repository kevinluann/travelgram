# Travelgram

**Página única** de perfil de viagens, desenvolvida com HTML5 e CSS3.

## 🎯 Objetivo

Projeto de estudo para praticar:
- HTML5 semântico
- CSS3 com variáveis customizadas
- Flexbox para layouts
- Organização de arquivos CSS

## Estrutura da Página

- **Header** com navegação principal
- **Perfil** com foto, bio e estatísticas
- **Galeria** com 12 fotos de viagens
- **Footer** com copyright e links legais

## Tecnologias
- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e layout

## Estrutura do Projeto

```
Travelgram/
├── assets/
│   ├── icons/          # Ícones SVG
│   └── images/         # Fotos de perfil e galeria
├── styles/
│   ├── global.css      # Variáveis e estilos globais
│   ├── header.css      # Estilos do cabeçalho
│   ├── about.css       # Estilos da seção sobre
│   ├── gallery.css     # Estilos da galeria
│   └── footer.css      # Estilos do rodapé
├── index.html          # Página principal
└── README.md          # Este arquivo
```
